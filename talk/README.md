# kube-rs Talk

Presentation materials for "Extend Kubernetes Using Rust: A Quick Introduction to kube-rs".

## Contents

- `kube-rs.typ` - Typst source file for the presentation
- `kube-rs.pdf` - Compiled presentation slides
- `assets/` - Images, diagrams, and graphics used in slides
- `archive/` - Previous versions of the presentation

## Building

The presentation is built using [Typst](https://typst.app/):

```bash
typst compile kube-rs.typ
```
